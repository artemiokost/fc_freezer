# ❄️ EA SPORTS FC 26 — Private Kernel AI Freezer v4.0

Ультимативный приватный трейнер Ring 0 с беспроводной архитектурой ядра операционной системы и интерактивным TUI-интерфейсом на базе библиотеки **Ratatui**. Позволяет мгновенно парализовать логику ИИ ботов противника в оффлайн-матчах (Карьера, Быстрый матч, Турниры) без изменения файлов игры и без использования капризных DLL-модификаций.

---

## 🚀 Ключевые Особенности

* **Kernel Hooking (Ring 0):** Комплекс общается с ядром «по воздуху» через легальный системный вызов `NtQuerySystemInformation`. Драйвер перехватывает системную таблицу вызовов по магическому коду `0x777FFFFF`, полностью скрывая факт записи в память от античита EAAC.
* **Игра без сворачиваний (Global Hotkeys):** Включение и выключение заморозки привязано к глобальной горячей клавише **`F5`**. Чит считывает нажатие прямо внутри матча, издавая звуковой сигнал подтверждения (`Beep`).
* **Интерактивный UI (Ratatui + Crossterm):** Красивая киберпанк-панель управления в реальном времени отображает PID игры, статус поиска сигнатуры Frostbite, текущее состояние удержания нулей и системные логи.
* **Автоматический JSON-конфиг:** При первом запуске сканер создает файл `config.json`. Если игра обновится, вам больше не нужен компилятор — достаточно просто изменить байты сигнатуры в текстовом файле, и трейнер подхватит их на лету.
* **Спецификация Rust 2024:** Весь низкоуровневый код (`no_std`) и FFI-блоки полностью оптимизированы под жесткие требования безопасности компилятора Rust 2024 без предупреждений.

## 🛠️ Инструкция по Чистой Сборке (в Git Bash)

Сборка всего воркспейса и линковка системного файла ядра Windows выполняется тремя командами в терминале **MINGW64**:

### 1. Компиляция всего проекта в Релиз
```bash
# Находясь в корне проекта fc_freezer, собираем драйвер и сканер
cargo build --release
```

### 2. Линковка Kernel-драйвера через MSVC link.exe
```bash
# Переходим в папку с результатами сборки Cargo
cd ~/.cargo-target/release

# Вызываем оригинальный линковщик Microsoft C++ (одной строкой)
MSYS_NO_PATHCONV=1 "/c/Program Files/Microsoft Visual Studio/2022/Community/VC/Tools/MSVC/14.44.35207/bin/Hostx64/x64/link.exe" /DRIVER /SUBSYSTEM:NATIVE /ENTRY:DriverEntry /NODEFAULTLIB /OUT:fc_driver.sys fc_driver.lib
```

### 3. Перенос боевого комплекта в Пусковую папку
```bash
# Переносим файлы в изолированную зону исключений антивируса
cp fc_driver.sys /c/FreezerLoader/
cp fc_freezer.exe /c/FreezerLoader/
```

## 🎮 Алгоритм Использования и Запуск

Следование этой пошаговой инструкции гарантирует 100% стабильность работы без падений в синий экран (BSOD) и ошибок блокировки драйверов:

### Шаг 1. Подготовка Системы (Выполняется 1 раз)
Убедитесь, что папка `C:\FreezerLoader\` добавлена в исключения Защитника Windows, а аппаратная изоляция ядра (HVCI/VBS) и Secure Boot отключены в BIOS вашей материнской платы AMD. В `msinfo32` статус безопасности должен быть `Не включено`.

### Шаг 2. Загрузка Драйвера в Ring 0 (ДО запуска игры)
1. Нажмите Пуск, введите **`cmd`** и откройте **Командную строку от имени Администратора**.
2. Выполните принудительный маппинг кода в свободные физические страницы памяти:
   ```cmd
   cd C:\FreezerLoader
   kdmapper_Release.exe fc_driver.sys
   ```
3. Убедитесь, что KDMapper вывел финальные строки `DriverEntry returned 0x0` и **`[+] success`**.
4. **Полностью закройте черное окно `cmd`**. Маппер завершил работу, а ваш хук остался скрытно функционировать в ядре ОС.

### Шаг 3. Включение Трейнера и Игра
1. Зайдите в папку `C:\FreezerLoader\` через обычный Проводник и запустите файл **`fc_freezer.exe`** (двойным кликом). Развернется графическое меню Ratatui.
2. Запустите игру **EA SPORTS FC 26** абсолютно стандартным способом через ваш лаунчер. Трейнер в фоне поймает PID игры.
3. Начните матч в режиме Карьеры. Как только футболисты появятся на поле и прозвучит свисток арбитра, сканер автоматически найдет адрес ИИ Frostbite по сигнатуре и обновит статус.
4. **Управляйте читом прямо в матче:**
    * Нажмите клавишу **`F5`** на клавиатуре — компьютер издаст высокий писк, рамка в консоли загорится **зеленым**, а боты соперника намертво застынут на месте.
    * Нажмите клавишу **`F5`** еще раз — компьютер издаст низкий писк, рамка в консоли загорится **красным**, и удержание памяти снимется — боты продолжат обычную игру.
    * Для закрытия трейнера нажмите клавишу **`Q`** внутри окна консоли.

## ⚙️ Структура файлов в `C:\FreezerLoader\`

Для успешной работы в пусковой папке должен находиться следующий боевой комплект:
* 📄 `kdmapper_Release.exe` — Инструмент ручного маппинга в обход цифровых подписей.
* 📄 `fc_driver.sys` — Скомпилированный Kernel-код на Rust для управления физической памятью.
* 📄 `fc_freezer.exe` — Графический TUI-сканер сигнатур и перехватчик горячих клавиш.
* 📄 `config.json` — Автоматически создаваемый текстовый файл конфигурации паттерна байт.

```json
{
  "pattern_bytes": [ 72, 139, 5, -1, -1, -1, -1, 72, 139, 136, 139, 1, 0 ]
}
```
*(Число `-1` в конфигурационном файле JSON выступает в роли маски `??` для динамических смещений памяти).*
