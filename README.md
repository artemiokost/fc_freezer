# ❄️ EA SPORTS FC 26 — Private Kernel Match Manager v4.5

Ультимативный приватный мульти-хак Ring 0 с беспроводной архитектурой ядра операционной системы и интерактивным TUI-интерфейсом на базе библиотеки **Ratatui** в стиле коммерческого софта **CheatArmy**.

---

## 🛠️ Автоматическая Сборка «В один клик» (Java-Style Automation)

Для сборки, линковки и копирования комплекса больше не нужно использовать консоли и вводить команды вручную. Процесс полностью автоматизирован через механизм интеграционных тестов в среде разработки IntelliJ IDEA:

1. Откройте в дереве проекта файл: `fc_freezer` 📂 -> `tests` 📂 -> 📄 **`build_pipeline.rs`**
2. Нажмите на **зелёную кнопку `Play` (Run)**, которую IntelliJ IDEA нарисовала слева от функции `fn build_and_link_cheat_arm()`.
3. Выберите пункт **`Run 'build_pipeline::build_and_link_cheat_arm'`**.

### Что произойдет автоматически:
* Вызовется `cargo build --release` для компиляции всех модулей воркспейса.
* Скрипт обратится в вашу кастомную папку таргета `C:\Users\artem\.cargo-target\release\`.
* Запустится линковщик Visual Studio MSVC и соберёт системный файл ядра `fc_driver.sys`.
* Готовый комплект (`fc_driver.sys` и `fc_freezer.exe`) принудительно перенесётся в пусковую папку `C:\FreezerLoader\`.

## 🎮 Алгоритм Использования и Горячие Клавиши

Следование этой пошаговой инструкции гарантирует 100% стабильность работы без падений в синий экран (BSOD) и ошибок блокировки драйверов:

### Шаг 1. Загрузка Драйвера в Ring 0 (ДО запуска игры)
1. Нажмите Пуск, введите **`cmd`** и откройте **Командную строку от имени Администратора** (Не PowerShell, не Git Bash).
2. Выполните принудительный маппинг кода в свободные физические страницы памяти процессора AMD:
   ```cmd
   cd C:\FreezerLoader
   kdmapper_Release.exe fc_driver.sys
   ```
3. Убедитесь, что KDMapper вывел финальные строки `DriverEntry returned 0x0` и **`[+] success`**.
4. **Полностью закройте черное окно `cmd`**. Маппер завершил работу, а ваш беспроводной хук Ring 0 остался скрытно функционировать в ядре ОС.

### Шаг 2. Запуск Трейнера и Горячие Клавиши прямо в матче
1. Зайдите в папку `C:\FreezerLoader\` через обычный Проводник и запустите файл **`fc_freezer.exe`** (двойным кликом). Развернется графическое меню Ratatui.
2. Запустите игру **EA SPORTS FC 26** абсолютно стандартным способом через ваш лаунчер. Трейнер в фоне поймает PID игры.
3. Перейдите в онлайн или оффлайн сессию. Как только начнется матч на поле, сканер автоматически найдет сетевые и игровые дескрипторы.
4. **Управляйте читом прямо внутри игры без сворачиваний:**
   * **`[F5]`** — Полный паралич ИИ ботов соперника (Disable AI Opponent)
   * **`[F6]`** — Включение Division Spoofer (Уровень подменяется из файла `config.json`)
   * **`[F7]`** — Draft Round Modifier (Модификатор раундов драфта 0–3)
   * **`[F8]`** — WL Win Record Spoofer (Подмена побед в FUT Champs)
   * **`[F9]`** — Server Changer (Подмена дескриптора локации сетевого шлюза)
   * **`[F10]`** — ALT-TAB Background Bypass (Блокировка перехвата фокуса и дисконнектов)
   * **`[Q]`** — Нажмите внутри окна консоли трейнера для безопасного закрытия.

## ⚙️ Динамическая Настройка через `config.json`

При первом запуске трейнер автоматически создаст в папке `C:\FreezerLoader\` текстовый файл конфигурации. Чтобы изменить целевой дивизион, раунд драфта или количество побед в Weekend League без перекомпиляции кода, просто отредактируйте параметры в Блокноте:

```json
{
   "pattern_ai_freeze": [72, 139, 5, -1, -1, -1, -1, 72, 139, 136, 139, 1],
   "pattern_network_data": [139, 5, -1, -1, -1, -1, 0, 0],
   "spoofed_division": 1,
   "spoofed_draft_round": 3,
   "spoofed_wl_wins": 15,
   "server_location_id": 14
}
```
*(Число `-1` выступает в роли маски `??` для динамических смещений памяти).*
